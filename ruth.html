Basic Facts

  • Common Name: Red-Eyed Tree Frog
  • Scientific Name: Agalychnis callidryas
  • Type: Amphibians
  • Average Life Span: 5 years

Fun Facts

  • Red eyed tree frogs are nocturnal
  • Being green helps them camouflage within the trees
  • Lego created a red eyed tree frog set
  • The red eyed tree frog is not poisonous
